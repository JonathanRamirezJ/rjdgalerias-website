## RJD - Galerias WebbApp with React JS

## Quick Start

1. `yarn install`
2. `yarn run serve` to develop (open your browser at [http://localhost:3000])
3. `yarn run build` to create a distribution build

## Requirements

* [node.js](https://nodejs.org) v6.2.x (LTS)
